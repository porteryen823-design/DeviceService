🔌 MQTT 連線測試頁面

📋 測試說明

這個頁面用於測試當訪問 http://127.0.0.1:8080/ProxyStatus 時的 MQTT 自動連線功能。

測試步驟：

  1. 開啟瀏覽器開發者工具的主控台
  2. 訪問 http://127.0.0.1:8080/ProxyStatus
  3. 觀察主控台中的 MQTT 連線日誌
  4. 檢查頁面上的連線狀態指示器
🔌 MQTT 狀態：未連線

📝 連線日誌

🔧 環境設定

MQTT 設定：

  • 主機：localhost
  • 埠號：8093
  • 協定：ws
  • 用戶端 ID：frontend_mqtt_client_XX

預期行為：

  • 當訪問 /ProxyStatus 路由時自動連線 MQTT
  • 在瀏覽器主控台顯示詳細的連線日誌
  • 頁面上顯示連線狀態指示器
  • 離開頁面時自動斷開連線
